# 雑学王 (Zatsugaku King)

雑学クイズで楽しく学べる Android アプリです。

## 主な機能

- **ホーム** … クイック挑戦でランダム5問のクイズを開始。
- **クイズ** … 毎回ランダム出題。回答後すぐに正解／不正解を表示。正解でEXP獲得、連続正解・全問正解でボーナス。
- **知識センター** … ジャンル別に学べる（すべて・日本・世界・IT・アニメ・食べ物・歴史など）。
- **ランキング** … 累計正答数で順位表示。
- **マイページ** … ニックネーム・アイコン変更、言語切替（日本語・英語・中国語）、累計EXP・正解数・連続正解の確認。レベル（Lv.1〜10）に応じて称号とアバターを解放。

- Kotlin / ViewBinding / ViewModel + Fragment
- 多言語対応（日本語・英語・中国語）
